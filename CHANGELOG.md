# История изменений

Все значимые изменения в этом проекте будут документироваться в этом файле.

## [2.1.1] - 2024-XX-XX

### Исправлено

- Исправлена проблема с горением игроков во время дождя/шторма - теперь игроки не горят и не получают эффекты солнечного света во время осадков

## [2.1.0] - 2024-XX-XX

### Добавлено

- Поддержка Minecraft 1.21.1
- Команда `/dwarfmode info` для просмотра информации о плагине
- Команда `/dwarfmode tear <amount>` для получения слез гаста
- Настройка `checkInterval` для настройки интервала проверки игроков
- Поддержка медной шляпы (COPPER_HELMET) для защиты от солнца
- Настройка `helmetProtection` для защиты от урона от солнца шлемами/головами
- Настройка `skyBlindness` для слепоты на открытом небе
- Настройка `surfacePenalties` для штрафов на поверхности
- Настройка `damageOverTime` для урона со временем на поверхности
- Поддержка настройки разрешенных миров через `enabledWorlds`
- Бонусы под землей с настраиваемыми эффектами по глубине
- Штрафы за солнечный свет с настраиваемыми эффектами
- Оптимизированная система проверки игроков через периодические задачи

### Изменено

- Улучшена производительность за счет использования периодических проверок вместо событий движения
- Улучшена система применения эффектов под землей (накопительные эффекты)
- Обновлена система защиты от солнца (поддержка больше типов шлемов и голов)

### Исправлено

- Исправлена проблема с обновлением эффектов на клиенте
- Исправлена система применения эффектов под землей (правильное накопление)

## [2.0.0] - 2024-XX-XX

### Добавлено

- Базовая функциональность плагина
- Система бонусов под землей
- Система штрафов за солнечный свет
- Система штрафов на поверхности
- Конфигурационный файл с подробными комментариями
- Команда `/dwarfmode reload` для перезагрузки конфигурации
