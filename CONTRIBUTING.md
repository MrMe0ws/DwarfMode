# Руководство по внесению вклада

Спасибо за интерес к проекту DwarfMode! Мы приветствуем вклад от всех.

## Как внести вклад

### Сообщение об ошибках

Если вы нашли ошибку:

1. Проверьте, не была ли она уже зарегистрирована в [Issues](../../issues)
2. Если нет, создайте новый Issue с подробным описанием
3. Приложите логи сервера и конфигурацию (без чувствительных данных)

### Предложение новых функций

Если у вас есть идея для новой функции:

1. Проверьте, не была ли она уже предложена
2. Создайте новый Issue с описанием функции
3. Объясните, почему эта функция будет полезной

### Pull Requests

1. Сделайте Fork проекта
2. Создайте ветку для вашей функции (`git checkout -b feature/AmazingFeature`)
3. Внесите изменения
4. Убедитесь, что код компилируется без ошибок
5. Закоммитьте изменения (`git commit -m 'Add some AmazingFeature'`)
6. Запушьте в ветку (`git push origin feature/AmazingFeature`)
7. Откройте Pull Request

## Стандарты кода

- Следуйте стилю кода, используемому в проекте
- Добавляйте комментарии к сложным участкам кода
- Обновляйте документацию при необходимости
- Тестируйте изменения перед отправкой PR

## Вопросы?

Если у вас есть вопросы, создайте Issue с меткой `question`.
